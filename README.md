# 🤖 Utbaa AI

Utbaa AI is a modern AI chatbot web application designed and developed by **Muhammad Utbaa**, a Cambridge Class 7 student passionate about technology, robotics, and innovation.

It is inspired by advanced AI assistants like ChatGPT and built to provide a smooth, fast, and intelligent conversation experience.

---

## 🚀 Features

- 💬 Smart AI chat assistant (real-time responses)
- 🎨 Clean, modern, and relaxing UI
- 🧠 Multi-mode system (Chat, Coding Help, Learning Support)
- 📚 Homework & study assistance (Math, Science, English)
- 🧑‍💻 Code Doctor mode (bug fixing + explanations)
- 🧾 Conversation memory (personalized responses)
- ⚡ Smooth streaming responses
- 🔍 Built-in idea and problem-solving support

---

## 🛠️ Tech Stack

- Frontend: HTML / CSS / JavaScript (or React depending on build)
- Backend: Node.js / Express
- AI Integration: OpenAI-powered API system
- Hosting: Replit / Vercel / Render compatible
